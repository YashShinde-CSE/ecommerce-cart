# E-Commerce Cart

A simple shopping cart application built using HTML, CSS, and JavaScript that allows users to add and remove products while calculating the total purchase amount.

## Features

- Display available products
- Add products to cart
- Remove products from cart
- Automatic total price calculation
- User-friendly interface
- Dynamic cart updates using JavaScript

## Technologies Used

- HTML
- CSS
- JavaScript

## Concepts Implemented

- DOM Manipulation
- Event Handling
- Arrays and Objects
- Dynamic Content Rendering
- Shopping Cart Logic
